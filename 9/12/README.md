<h3>मोघाशा मोघकर्माणो मोघज्ञाना विचेतसः |<br/>राक्षसीमासुरीं चैव प्रकृतिं मोहिनीं श्रिताः ||९-१२||</h3>
<pre>moghāśā moghakarmāṇo moghajñānā vicetasaḥ .<br/>rākṣasīmāsurīṃ caiva prakṛtiṃ mohinīṃ śritāḥ ||9-12||</pre>
