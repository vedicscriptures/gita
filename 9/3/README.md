<h3>अश्रद्दधानाः पुरुषा धर्मस्यास्य परन्तप |<br/>अप्राप्य मां निवर्तन्ते मृत्युसंसारवर्त्मनि ||९-३||</h3>
<pre>aśraddadhānāḥ puruṣā dharmasyāsya parantapa .<br/>aprāpya māṃ nivartante mṛtyusaṃsāravartmani ||9-3||</pre>
