<h3>दम्भो दर्पोऽभिमानश्च क्रोधः पारुष्यमेव च |<br/>अज्ञानं चाभिजातस्य पार्थ सम्पदमासुरीम् ||१६-४||</h3>
<pre>dambho darpo.abhimānaśca krodhaḥ pāruṣyameva ca .<br/>ajñānaṃ cābhijātasya pārtha sampadamāsurīm ||16-4||</pre>
