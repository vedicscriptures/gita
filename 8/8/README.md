<h3>अभ्यासयोगयुक्तेन चेतसा नान्यगामिना |<br/>परमं पुरुषं दिव्यं याति पार्थानुचिन्तयन् ||८-८||</h3>
<pre>orsaṃśayama abhyāsayogayuktena cetasā nānyagāminā .<br/>paramaṃ puruṣaṃ divyaṃ yāti pārthānucintayan ||8-8||</pre>
