<h3>चेतसा सर्वकर्माणि मयि संन्यस्य मत्परः |<br/>बुद्धियोगमुपाश्रित्य मच्चित्तः सततं भव ||१८-५७||</h3>
<pre>cetasā sarvakarmāṇi mayi saṃnyasya matparaḥ .<br/>buddhiyogamupāśritya maccittaḥ satataṃ bhava ||18-57||</pre>
