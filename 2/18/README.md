<h3>अन्तवन्त इमे देहा नित्यस्योक्ताः शरीरिणः |<br/>अनाशिनोऽप्रमेयस्य तस्माद्युध्यस्व भारत ||२-१८||</h3>
<pre>antavanta ime dehā nityasyoktāḥ śarīriṇaḥ .<br/>anāśino.aprameyasya tasmādyudhyasva bhārata ||2-18||</pre>
