<h3>अव्यक्तादीनि भूतानि व्यक्तमध्यानि भारत |<br/>अव्यक्तनिधनान्येव तत्र का परिदेवना ||२-२८||</h3>
<pre>avyaktādīni bhūtāni vyaktamadhyāni bhārata .<br/>avyaktanidhanānyeva tatra kā paridevanā ||2-28||</pre>
