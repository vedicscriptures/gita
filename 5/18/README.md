<h3>विद्याविनयसम्पन्ने ब्राह्मणे गवि हस्तिनि |<br/>शुनि चैव श्वपाके च पण्डिताः समदर्शिनः ||५-१८||</h3>
<pre>vidyāvinayasampanne brāhmaṇe gavi hastini .<br/>śuni caiva śvapāke ca paṇḍitāḥ samadarśinaḥ ||5-18||</pre>
